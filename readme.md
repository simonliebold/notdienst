## Employments
### Get all employments
<pre>GET /employments</pre>

### Get one employment
<pre>GET /employments/:id</pre>

### Create one employment
<pre>POST /employments</pre>

### Update one employment
<pre>PUT /employments/:id</pre>

### Delete one employment
<pre>DELETE /employments/:id</pre>
